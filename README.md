# PortfolioHUB

PortfolioHUB é uma página central para organizar currículo, projetos, apresentação e documentação acadêmica de Carlos Vinícius Ferreira Morgado.

## Objetivo

Centralizar os materiais do portfólio em um site simples, versionado no GitHub e pronto para publicação no GitHub Pages.

## Estrutura

- `index.html`: página principal do PortfolioHUB.
- `assets/`: arquivos de estilo e script.
- `docs/`: documentação de segurança e colaboração.
- `Curriculo.docx`: currículo acadêmico/profissional.
- `portfolioHUB.pptx`: apresentação do portfólio.
- `entrega/`: relatório final e arquivos de apoio para envio.

## Segurança

O site foi montado como página estática, sem banco de dados, login ou coleta de dados pessoais.
